# NIVORA

**Graduate Learning, evolved.**

Plataforma de posgrado interactivo que convierte materiales académicos en rutas de dominio, práctica activa, evaluaciones y documentos de estudio.

## Inicio rapido

```bash
npm install
npm run dev
```

## Verificacion

```bash
npm run lint
npm run validate:vault
npm run build
npm run preview
```

La plataforma funciona completamente con datos mock locales. No incluye backend, API keys ni conexión real a Supabase.
